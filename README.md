# Students-Data-analysis-in-R-programming-
Student Data Analysis and Data Cleaning in R

This project demonstrates how raw student data can be transformed into meaningful insights using base R. It involves cleaning inconsistent categorical data, merging multiple datasets, and handling missing values to create a reliable analytical dataset.

Key tasks include data preprocessing using substring extraction, dataset integration using merge operations, feature engineering, and the creation of categorical performance metrics. A custom function was also developed to classify student performance based on final marks.

This project highlights practical skills in data manipulation, data quality improvement, and analytical thinking using core R programming techniques.
